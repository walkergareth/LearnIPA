Symbols for the phonemic transcription of English

This page contains a list of symbols for use in the phonemic transcription of English along with keywords and audio illustrations. If no audio controls appear check whether your browser supports the audio element and if not, try a browser which does. If IPA symbols seem to be absent this is probably because your browser cannot access a Unicode font: see this page for advice on installing and using such fonts.
