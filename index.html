Learning the International Phonetic Alphabet

This site provides access to various resources to help with learning the sounds of the International Phonetic Alphabet (IPA).

System requirements

Every reasonable effort has been made to ensure that these pages work on a wide range of computer set-ups and mobile devices. In the event of unexpected behaviour on a mobile device, try accessing this site on a computer.

To make good use of the pages you will need a font installed which provides IPA symbols. If IPA symbols seem to be absent this is probably because your browser cannot access a Unicode font: see this page for advice on installing and using such fonts.

If no videos appear where you expect them, check whether your browser supports the video element and if not, try a browser which does. If no audio controls appear where you expect them, check whether your browser supports the audio element and if not, try a browser which does.
